Artistic Workshops - Bug Report

Resolved Issues & Technical Fixes

Fixed النص المختفي (Invisible Hero Text)

المشكلة: عناوين القسم الأول (Hero Section) كانت مختفية تماماً.
السبب: استخدمنا كود animation: fadeUp في الـ CSS، لكن نسينا نعرف الـ @keyframes fadeUp نفسها. المتصفح كان مستني التعليمات ومظهرش النص.
الحل: أضفنا تعريف الحركة الناقص في ملف الـ CSS.
The Issue: The Hero Section titles were completely invisible.
The Cause: We applied animation: fadeUp in the CSS but failed to define the @keyframes fadeUp rule. The browser hid the text waiting for animation instructions that never existed.
The Fix: We added the missing keyframe definition to the CSS file.
The Fix (Added to workshops.css): 
@keyframes fadeUp {
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

Fixed اختفاء الملمس الحرفي (Missing Artisan Texture)

المشكلة: خلفية الصفحة كانت سادة ومفيهاش "الملمس المحبب" (Grain Texture) اللي بيميز تصميم Artistic.
السبب: الـ CSS كان بيطبق الملمس ده على كلاس اسمه .workshop-page، لكن تاج الـ <body> في ملف الـ HTML مكانش واخد الكلاس ده.
الحل: أضفنا class="workshop-page" لتاج الـ body.
The Issue: The page background was flat and missing the signature "Grain Texture" overlay.
The Cause: The CSS applied this texture to the .workshop-page class, but the <body> tag in the HTML file was missing this specific class name.
The Fix: We added class="workshop-page" to the body tag.
The Fix (workshops.html): 
<!-- Before -->
<body id="top">

<!-- After -->
<body id="top" class="workshop-page">
